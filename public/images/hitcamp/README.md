# Hit Camp image assets

Drop the supplied PNGs here. They are used as **CSS backgrounds only** (text
stays live HTML) with dark gradient overlays, subtle hover zoom, and responsive
cropping. Missing files degrade gracefully to the dark studio background — the
app will not crash if an asset is absent.

## Required files

```
header.png          # Homepage hero
production.png
songwriting.png     # used by the Creativity discipline
music-theory.png
ear-training.png
mixing.png
beatmaking.png
sound-design.png
arrangement.png
lyrics_melody.png
recording.png
music-business.png
```

Do not bake text into the images — all headings and copy are rendered as HTML
on top of these backgrounds.
